# Protocol Core

Core shared models and types for the Silica Protocol.

## Crates

- **rs-models** (`silica-models`) - Rust data models, types, and cryptographic primitives
- **ts-models** - TypeScript type definitions (generated from Rust)

## Usage

### Rust

Add to your `Cargo.toml`:

```toml
[dependencies]
silica-models = { git = "https://github.com/Silica-Protocol/protocol-core", branch = "main" }
```

### TypeScript

```bash
npm install @silica-protocol/models
```
